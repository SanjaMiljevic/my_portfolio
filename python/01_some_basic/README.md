# String Practice

This folder contains beginner-level Python scripts for practicing
string manipulation concepts.

## Included scripts
- count_vowels.py – counts vowels in a string
- count_words.py – counts words while ignoring punctuation
- is_palindrome.py – checks if a string is a palindrome
- reverse_string.py – reverses a given string

## Concepts covered
- string methods
- slicing
- dictionaries
- input validation
- functions and docstrings
